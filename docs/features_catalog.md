# Features Catalog - NimaStartupSocial

## Core Features

- **Dynamic Shortcut Management**: Add, edit, remove, and reorder website shortcuts.
- **Smart Cataloging**: Automatically suggest Arabic/English names and icons based on URL (e.g., GitHub, YouTube, LinkedIn).
- **Custom Browser Resolution**: Specify which browser (system default, Chrome, Firefox, Edge, etc.) should open each specific shortcut.
- **RTL Support**: Full Right-to-Left layout for Arabic users.
- **Auto-Start**: Manage the application to start automatically with the operating system (Windows, Linux, macOS).
- **Import/Export**: Backup or share shortcut lists via JSON/CSV files.
- **Search & Filter**: Quickly find shortcuts by name or URL.

## UI/UX Features

- **Modern Atomic Design**: Consistent widget styles with clear hover and pressed states.
- **Circular Icon Badges**: Color-coded badges with initials or logos for quick visual identification.
- **Interactive Drag-and-Drop**: Reorder shortcuts directly in the management dialog.
- **Toast Notifications**: Non-blocking feedback for success messages.
